dlfcn.h

  • dlopen() - load a shared object (code compiled with -shared flag)
    void *addso = dlopen("./add.so", RTLD_LAZY);
  • dlsym() - load a method from loaded shared object
    binFunc add = (binFunc)dlsym(addso, "add");
  • dlclose() - dispose loaded shared object
    dlclose(addso);
